# 🔥 Campfire Pune 2026

Welcome to the official repository for the **Campfire Pune** website! 

Campfire Pune is an exciting 12-hour game jam where developers, designers, and creative minds come together to build innovative games. This landing page was built to share our event details, showcase the amazing open-source games built by our participants, and host our favorite memories from the day.

## 🚀 Live Demo
https://dotcomronak.github.io/campfire-pune

## 🛠️ Tech Stack
This website is built entirely with:
* Semantic **HTML5**
* Vanilla **CSS3** (Custom properties, CSS Grid, Flexbox)
* Vanilla **JavaScript** (Smooth scrolling, mobile menu toggling, FAQ accordion)
* Vibe-Coded? Yes, 60-70%
No heavy frameworks—just clean, lightweight, and responsive code!
